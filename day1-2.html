HTML Introduction

Tag, Element, dan Atribut

Tag di dalam dokumen HTML bertujuan untuk memberitahu browser bagaimana teks di dalamnya harus ditampilkan. Kebanyakan tag-tag dalam HTML ditulis secara berpasang semisal <"body> dan <"/body>. <"body> disini dinamakan tag pembuka dan <"/body> merupakan tag penutup.

Tag biasanya juga mengandung atribut (attribute), yaitu sebuah informasi tambahan yang diletakkan pada tag pembuka, misal <"p class=”pembuka”>…………<"/p> . Perlu dicatat bahwa setiap nama dan nilai setiap atribut memiliki fungsi yang berbeda-beda

Semua yang terdapat pada tag pembuka sampai tag penutup disebut dengan element. Misalnya begini, semua yang berada pada tag <'body> dan <'/body> dinamakan element body sedangkan <'body> dan <'/body> merupakan tag, bukan element.

Table di HTML

Tag Table di HTML ada 6 yaitu :

  1. Tag <'table> untuk membungkus tabelnya
  2. Tag <'thead> untuk membungkus bagian kepalanya
  3. Tag <'body> untuk membungkus bagian body tabel
  4. Tag <'tr> (table row) untuk membuat baris
  5. Tag <'td> (table data) untuk membuat cell
  6. Tag <'th> (table head) untuk membuat Judul pada header tabel

Form di HTML

Form di HTML dapat dibuat dengan tag <'form>, dan tag form memiliki 2 atribut wajib yaitu :

  • action, untuk menentukan aksi yang akan dilakukan setelah dikirim

Image, Video dan Audio

Images pada HTML

pada HTML untuk menampilkan image menggunakan tag <'img>

contoh image sebagai background ditaruh dalam tag div menggunakan attribut style

membuat image dengan link tag <'img> dibungkus menggunakan tag <'a>

Video pada HTML

Pada HTML untuk menampilkan video menggunakan tag <'video> dengan attribut controls kemudian tag <'source src="" type"video/mp4""> untuk source videonya beserta attribut src dan type videonya

Menampilkan Video dari internet menggunakan embed dengan tag <'iframe>

Audio dalam HTML

Pada HTML untuk menampilkan Audio menggunakan tag <'audio controls> dengan attribut controls kemudian tag <'source src="" type"audio/mp3""> untuk source audionya beserta attribut src dan type audionya
