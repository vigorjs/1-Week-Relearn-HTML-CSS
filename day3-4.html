Pengenalan Dasar CSS

Apa itu CSS?

CSS merupakan singkatan dari Cascade Style Sheet adalah sebuah bahasa untuk mengatur tampilan web sehingga terlihat lebih indah.

Tips belajar CSS

  1. Menghafalkan kode CSS (Sering2 Praktek)
  2. Menggunakan Inspect Element
  3. Menggunakan Visual Studio Code
  4. CSS Style Sheet

Selektor

Selektor adalah kata Kunci untuk memilih elemen HTML yang akan diberi style

Blok Deklarasi

tempat kita menuliskan attribut atau properti css yang akan diberikan kepada elemen yang telah diseleksi

Properti

Sekumpulan aturan yang akan diberikan didalam blok deklarasi kepada elemen yang dipilih

Nilai / Value

isi dari properti untuk merubah elemen HTML yang ingin diubah

Color pada CSS

Nilai warna pada css dibagi menjadi 5
  1. Nama warna (red,yellow,dll)
  2. Hexadesimal (#ffff)
  3. RGB & RGBA
  4. Hsl & Hsla
  5. Gradasi / Gradient
