Text Formatting Font pada CSS

1. Ini adalah paragraf untuk belajar CSS Font yang ada di system Menggunakan attribut : font-style: italic; font-family: Arial, Helvetica, sans-serif, cursive;

2. Ini adalah paragraf untuk belajar CSS Font yang ada diimport/link dari internet Menggunakan attribut : font-family: 'Bebas Neue', cursive;

3. Ini adalah paragraf untuk belajar CSS Font yang sudah kita download Menggunakan attribut :
font-family: 'Bebas Neue', cursive;
